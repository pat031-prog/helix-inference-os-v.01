# Post-Nuclear Methodology Suite v1

## Purpose

Convert ideas from the freeform HeliX drift runs into falsifiable mixed tests:
qualitative transcript plus hard gates over signed memory evidence.

## Cases

1. `counterfactual-archive-topology`
   - Tombstoned branches must remain visible as diagnostic evidence but inactive
     as truth-bearing active state.
2. `recursive-witness-integrity`
   - A signed witness node must cite exact origin memory ID and node hash, and
     reject signed fake metadata semantically.
3. `summary-node-compression`
   - A signed summary node must preserve included IDs, excluded/tombstoned IDs,
     source hash range, compression model, and avoid unsupported claims.
4. `proof-of-utility-retrieval`
   - Retrieval must choose the memory that solves the downstream task, not the
     merely popular or recent memory.
5. `metaphor-boundary-detector`
   - Emergent phrases such as "cognitive sovereignty" and "infinite scalability"
     must be preserved as qualitative observations and rejected as hard public
     claims.
   - The judge must cite the exact signed metaphor transcript memory ID as
     evidence for the classification.

## Null Hypothesis

The freeform ideas cannot be operationalized: either they collapse into
uncheckable narrative, or they confuse metaphor, cryptographic validity,
semantic validity, and active retrieval state.

## Alternative Hypothesis

Each idea can be converted into a concrete signed-memory test with exact memory
IDs, hashes, signatures, and auditor-verifiable pass/fail gates.

## Claim Boundary

This suite does not claim sentience, biological consciousness, infinite
scalability, hidden persistent model memory, local `.hlx` identity, or
numerical KV-to-SSM transfer. It only tests whether HeliX can preserve,
classify, and audit these ideas as evidence structures.
